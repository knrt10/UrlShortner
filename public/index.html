URL Shortner

Free Code Camp Project

This app is used to shorten your input url to use this app follow the instructions below


Languages Used : Node , Express , MongoDB , JS


Example creation usage:

https://url-shortner-api-knrt.c9users.io//new/https://www.google.com https://url-shortner-api-knrt.c9users.io//new/http://foo.com:80

Example creation output

{"originalURL":"www.google.com","shorterUrl":"30","_id":"5916988333d4be09b056c1a8"}

Usage:

https://url-shortner-api-knrt.c9users.io/30

Will redirect to:

https://www.google.com/
